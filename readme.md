## Learning React

Made this app to improve react concepts fetching data from github API.
It searches repositories and list their issues.
You can simply run:

yarn:

```
$ yarn start
```

npm:

```
$ npm run start
```

- Class Component;
- State and Props;
- Proptypes;
- Component lifecycle;
- Axios to connect to github API;
- React Router Dom;
- Styled-components;
- Localstorage;
